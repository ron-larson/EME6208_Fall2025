Project Reflection

This project for EME 6208, Introduction to Multimedia Development, was a valuable exercise in translating complex academic concepts into tangible visual designs. The assignment was split into two parts: creating a bitmap book cover using GIMP and a vector logo using Inkscape.

I used the book cover as an opportunity to explore my personal research interests in Computer Science Education. I designed a cover for a fictional book, Code in Translation: Bridging the Block-to-Text Divide, which addresses the "fragile learning" problem—the difficulty many novice programmers face when moving from visual, block-based coding (like Scratch) to syntax-heavy, text-based languages (like Python). The cover became a visual metaphor for this challenge, depicting a learner's journey from a colorful, familiar "Block World" to a distant, abstract "Text World." A luminous AI-powered bridge connects the two, symbolizing a potential pedagogical solution. This required sourcing three separate images and meticulously combining them using a wide array of GIMP's tools, including layer masking, colorization with Overlay blending, and clipping masks to build the scene from scratch.

For the second part, I designed a personal "AI Circuit Monogram" logo. Inspired by the integrated Roger Federer "RF" logo, I structurally merged my initials, "RL," into a single, balanced glyph. The letterforms are rendered as "connections" and simplified "nodes," directly reflecting my research in AI and neural networks. I used Inkscape to build this as a clean, scalable vector graphic, ensuring it would be functional for any professional use.

Ultimately, this project was a powerful lesson in semantic design. Every choice was intentional—from the Scratch-like colors used in the "Block World" to the cohesive purple color palette that links the logo and the cover. I learned how to use both bitmap and vector tools to not just create an image, but to tell a clear and meaningful story.
