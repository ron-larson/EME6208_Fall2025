This page captures the submissions for two projects: 1) an HTML and Javascript-based quiz, and 2) an HTML and CSS-based drag-and-drop assessment.

HTML and Javascript Quiz and Drag-and-Drop Assessment

This project was an exercise in deconstructing, modifying, and personalizing existing web code. The task was to take a base HTML and JavaScript quiz template and customize its structure, content, and visual design. For my project, I created the "Code Breakers: What is an Algorithm?" quiz, a 10-question assessment aimed at a 7th-grade audience. This topic was an opportunity to apply instructional design principles by scaffolding a foundational Computer Science concept for novice learners.

The modification process was comprehensive. In the JavaScript file (popsolution.js), I expanded the quiz structure by doubling the questions from five to ten, updating the var total variable, and carefully rewriting all questions, answers, and the corresponding solution array. In the HTML file (index.html), I customized the 'look and feel' by changing the page's background color (#fcba03) and adding a custom title and author caption to give the project a unique identity.

A significant challenge emerged during development. After adding my new questions, the quiz page rendered with blank questions. I used a systematic troubleshooting process: by adding each question back into the .js file one-by-one and re-testing, I isolated the bug to lines containing double quotes (") within the question text. The solution was to replace these with the JavaScript escape sequence (\"), which allowed the strings to be parsed correctly. This debugging phase was a key learning experience, reinforcing the importance of methodical testing and the precise syntax required by JavaScript. The entire project was edited in Notepad on Windows 11 and is hosted in a public GitHub repository.

The original script foundation for this quiz was provided by JavaScript Kit.
